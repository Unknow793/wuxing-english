# 狂暴模式 · 备选方案

> 用于更高难度的挑战模式（中级/高级），与方案A（双倍攻击）配合使用。

---

## 方案B：属性暴涨

**触发条件**：第6回合起

**效果**：
- Boss ATK × **1.5**（攻击更狠）
- Boss DEF × **1.5**（更难打动）
- 每回合开始回复 **5% HP**

**适合场景**：中级难度，作为方案A之后的第二层强化
**实现要点**：
- 在 `bossTurn()` 开始时检查是否狂暴激活，是则 `boss.atk *= 1.5; boss.def *= 1.5`
- 每回合恢复 `boss.maxHp * 0.05` 的HP
- 注意恢复HP值需要取整

---

## 方案C：综合惩罚

**触发条件**：第6回合起

**效果**：
- 答对奖励伤害从 **10 → 5**（答题磨血变难）
- 答错扣血 **×1.5**
- 每回合自动燃烧玩家 **1张手牌**（战场恶化）
- Boss 外观增加 🔥 特效提示

**适合场景**：高级难度，全面压迫
**实现要点**：
- `answerBossQuestion` 中正确时 `BATTLE.bossHp -= 5`（原本10）
- 答错时 `rawDmg *= 1.5`
- `bossTurn()` 末尾随机弃掉1张可用手牌
- 复用方案A已有的 CSS 动画

---

## 方案D：环境异变（最丰富）

**触发条件**：第6回合起，每3回合切换一种"狂暴形态"

| 形态 | 效果 | 图标 |
|------|------|------|
| 烈焰形态 | Boss ATK × 2 | 🔥 |
| 铁壁形态 | Boss DEF × 2 | 🛡️ |
| 混沌形态 | 答题选项从4个变为5个 | 💫 |
| 风暴形态 | 每回合弃1张手牌 | 🌪️ |

**形态循环**：烈焰 → 铁壁 → 混沌 → 风暴 → 烈焰 → ...

**适合场景**：终极挑战/BOSS连战模式
**实现要点**：
- `BATTLE.berserkForm` 记录当前形态索引（0-3）
- 每3回合切换一次：`bossTurn()` 中 check `if (BATTLE.roundCount % 3 === 0)`
- 各形态的效果在对应函数中分派
- 形态切换时显示过渡动画/文字提示

---

## 组合建议

| 难度级别 | 狂暴方案组合 |
|---------|-------------|
| 初级 (150HP) | 方案A（双倍攻击） |
| 中级 (250HP) | 方案A + 方案B（属性暴涨） |
| 高级 (400HP) | 方案A + 方案B + 方案C（综合惩罚） |
| 终极 (???HP) | 方案D（环境异变，含A/B/C随机触发） |
